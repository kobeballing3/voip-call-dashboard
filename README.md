A clean SaaS-style dashboard prototype for managing VoIP call campaigns, live call activity, and lead uploads.

## 🌐 Overview

VoIP Call Dashboard is a frontend prototype designed to demonstrate a simple campaign management interface for voice automation systems.

## ⚙️ Features

- Create call campaigns
- View live call status
- Lead upload placeholder
- Clean dark SaaS-style UI
- Simple dashboard layout

## 🧰 Tech Stack

- HTML
- CSS

## 📌 Status

Frontend prototype — ready for future backend/API integration.
